API Documentation (Mock)

Example REST endpoints for the Employee Directory Search system.

POST /api/register
  Body: { "name": "John Doe", "email": "john@company.com", "password": "xxxx" }

POST /api/login
  Body: { "email": "john@company.com", "password": "xxxx" }
  Response: { token: "JWT_TOKEN" }

GET /api/employees
  Query: ?page=1&limit=10&q=alice&department=Engineering
  Response: {
    data: [ { _id, name, email, department, position, phone } ],
    page, total
  }

GET /api/employees/:id
  Response: { _id, name, email, department, position, phone, skills, managerId }

POST /api/employees  (Admin only)
  Body: { name, email, department, position, phone, skills }

PUT /api/employees/:id  (Admin only)
DELETE /api/employees/:id  (Admin only)
